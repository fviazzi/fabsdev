Project

Stim Social

This is my biggest project, working on it for the last 5 years. It is a SAAS product for a company based on Salt Lake City, Utah.

My main role on this project is solo frontend developer. Most of the software communicates with the backend through a REST API over JSON.

My work here consisted on making the UI of a dashboard with +26 unique views, for customers to create their Stim accounts, add their social media accounts (instagram & tiktok) and manage them through a series of targets, settings and several reports.

The design was already done when I joined, so I only had to create new components when needed, using the visual scheme that was already done.

This web app is based on a Opencart installation, with Paypal, Braintree and Stripe integrations for payments.

After 2 years of working on this project, I was required to find a way to create a cloned version of the app so new partners could have their own branded version of it.
This was made using Grunt and Node; running a node command, the default theme would be cloned into a directory using the new brand name, with customization capability and getting new features and changes from the default theme as they were are added.

This project is on maintenance since August, 2020.

Project

Lana Indumentaria

I co-own this website with my sister. This is a fashion ecommerce platform built in wordpress.

I use a base repo created by me, which brings a grunt configuration with a css preprocesor (LESS with autoprefixer), JS+ to JS 5 transpiling, image compresion (tinyimg) and minification (CSS & JS) integrated.
This repository also comes with a speed optimized theme (astra) as a base and it get's improved and customized by myself, integrating several marketing tools and keeping a region-optimized speed.

For this project, I've setted up an AWS EC2 instance, running ubuntu server 2016 with apache, php and mysql. Emails are set up using a combination of gmail with an external service (ImprovMX).

Payments are made with woocommerce and a local payment gateway. It allows credit cards, bank wires and "cash" (by printing a payment ticket).

It is integrated with Facebook shop, facebook pixel, google ads and mailchimp.
